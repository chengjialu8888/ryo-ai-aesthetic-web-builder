# Design Inspiration Library

## 1. Variant (variant.com/community)

**Best for**: UI style variations, component inspiration

**Use when**:
- Exploring different visual directions
- Need component-level inspiration
- Looking for trending styles

**Search patterns**:
- By component type (buttons, cards, forms)
- By style (minimal, bold, playful)
- By industry (SaaS, fintech, consumer)

---

## 2. Mobbin (mobbin.com)

**Best for**: Real app patterns, 611K+ screens

**Use when**:
- Need proven UX patterns
- Researching competitor flows
- Looking for specific screen types

## 4. Impeccable (impeccable.style)

**Best for**: Design system commands, principles

**Key Commands**:
1. **Balance** - Visual weight distribution
2. **Contrast** - Hierarchy through difference
3. **Proximity** - Grouping related elements
4. **Alignment** - Creating order
5. **Repetition** - Consistency and rhythm
6. **White Space** - Breathing room
7. **Typography** - Type as design
8. **Color** - Emotional communication
9. **Scale** - Size relationships
10. **Visual Hierarchy** - Guided attention
11. **Consistency** - Pattern recognition
12. **Simplicity** - Remove the unnecessary
13. **Functionality** - Form follows function
14. **Accessibility** - Design for all
15. **Responsiveness** - Adapt to context
16. **Loading Speed** - Performance matters
17. **User-Centric** - People first
18. **Feedback** - Confirm actions
19. **Flexibility** - Handle edge cases
20. **Innovation** - Push boundaries
