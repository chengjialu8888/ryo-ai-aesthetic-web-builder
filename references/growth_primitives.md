# Growth Primitives - Julian Shapiro + Brian Balfour

## Julian Shapiro - Demand Curve

### Core Formula
```
Purchase Rate = Desire − (Labor + Confusion)
```

**Maximize Desire**:
- Strong value proposition
- Emotional connection
- Social proof
- Scarcity/urgency (when authentic)

**Minimize Labor**:
- Clear CTA placement
- Short forms
- One-click actions
- Progressive disclosure
- Smart defaults

**Minimize Confusion**:
- Single primary action per screen
- Clear hierarchy
- Familiar patterns
- Remove jargon
- Visual guidance

### A/B Test Framework - 3 Hero Variants

#### Variant 1: Bold Claim
**Formula**: "The [product] that [competitors] don't want you to know about"

**When to use**:
- Disruptive products
- Contrarian positioning
- Established market with pain points
- Technical sophistication

#### Variant 2: Objection Killer
**Formula**: "Finally, a [product] that [solves specific pain point] without [common objection]"

**When to use**:
- Skeptical audiences
- High switching costs
- Previous category disappointments
- Risk-averse buyers

#### Variant 3: Story
**Formula**: "When [persona] started using [product], [specific outcome happened in time frame]"

**When to use**:
- Complex products
- Emotional decisions
- Transformation stories
- Building trust through relatability

## Brian Balfour - Reforge

### Loops > Funnels

**Funnel**: Linear, one-time, requires constant new input
**Loop**: Cyclical, self-reinforcing, compound growth

### Three Growth Loop Types

#### 1. Content Loop
```
User-generated content → SEO traffic → New users → More UGC
```

**Examples**: Stack Overflow, Yelp, Notion templates

#### 2. Product Loop
```
Usage → Value created → Sharing → New users → More usage
```

**Examples**: Figma, Loom, Typeform

#### 3. Community Loop
```
Engagement → Network effects → Retention → Advocacy → New users
```

**Examples**: Discord, Slack communities, Open source

### Retention = Growth

**Core Principle**: Growth without retention is a leaky bucket.

**Measurement**:
- D1/D7/D30 retention
- Activation rate (key action taken)
- Engagement frequency

## Content-Led SEO

### Topic Authority > Keywords

1. **Pillar Content**: Comprehensive guides (3000+ words)
2. **Cluster Content**: Specific subtopics linking to pillar
3. **Internal Linking**: Strategic connections
4. **Update Cycle**: Refresh quarterly

### Pain Point SEO

Target searches that indicate a problem:
- "How to [solve problem]"
- "Best [category] for [specific use case]"
- "[Alternative] vs [Competitor]"
- "Why is [problem] happening"
