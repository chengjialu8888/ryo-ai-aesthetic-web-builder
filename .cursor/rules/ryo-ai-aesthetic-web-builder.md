# Ryo AI Aesthetic Web Builder

## When to Apply
- Building AI product websites
- Creating landing pages
- Optimizing for conversion
- SEO strategy
- A/B testing frameworks

## Methodology

### Aesthetics Layer (Aesthetics of AI)
- 14 Visual Trends (Canvas×Signal, Playful Precision, etc.)
- 5 Brand Archetypes (Likeable Leaders, Bold Builders, etc.)
- Primitives-first design (color, type, layout, texture, motion)

### Conversion Layer (Julian Shapiro)
Formula: Purchase Rate = Desire − (Labor + Confusion)

Generate 3 Hero variants:
1. Bold Claim - Contrarian positioning
2. Objection Killer - Address pain points
3. Story - Emotional connection

### Growth Layer (Brian Balfour)
Loops > Funnels

Design one growth loop:
- Content Loop (UGC → SEO → Users)
- Product Loop (Usage → Value → Sharing)
- Community Loop (Engagement → Network → Advocacy)

### SEO Layer
- Content-led (topic authority)
- Pain-point driven
- Schema markup
- Internal linking

## Implementation Steps

1. Extract content primitives from user document
2. Select visual trends (2-3) + brand archetype (1)
3. Build conversion-focused information architecture
4. Create 3 Hero variants for A/B testing
5. Design growth loop integration
6. Implement with primitives-first CSS
7. Run QA checklist

## Quality Checklist
- [ ] 3 Hero variants created
- [ ] SEO meta tags complete
- [ ] Schema markup added
- [ ] Growth loop defined
- [ ] Accessibility checked
